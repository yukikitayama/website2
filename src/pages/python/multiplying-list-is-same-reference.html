Multiplying list is same reference

Jul 5, 2021

Issue

  • You just wanted to initialize matrix, but when you updated the elements, the result was not what you expected.
  • Other elements were updated with the same values.

Idea

  • When you multiply Python list, those lists all contain the same reference.
  • You need to use list comprehension instead.
