Template literals

  • Use backtick ` with ${} around your expression allows you to embed it in your string.

Function expression

  • Assign a function to a varialbe. The function does not have a function name (anonymous function).
  • To call the function, use the variable name with ().

All functions in JavaScript are methods

  • People use functions and methods interchangeably in JavaScript.
  • Methods are functions belonged to an object.
  • In JavaScript, everything is an object. Every function belongs to an object. So all the functions are methods.
  • A function which does not explicitly belong to an object belongs to window.
    • A function doStuff() could be written as window.doStuff();.
