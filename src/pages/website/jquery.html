jQuery

Jul 13, 2021

Idea

  • XXX.

Start by downloading

  • Go to https://jquery.com/
  • Click Download tab
  • Click Download the compressed, production jQuery X.X.X
  • Bunch of strings are shown in a browser, right click anywhere, save as, save as a JavaScript to a project folder.
  • Add script tag with src pointing at the JavaScript.
  • To test if that's working, window.jQuery object is true, if jQuery is loaded correctly, such as the following,
    • if (window.jQuery) { alert('jQuery is loaded.') }

Start by CDN

  • CDN means Content Delivery Network, donwloading jQuery from the internet when the page is loaded.
  • https://developers.google.com/speed/libraries#jquery
  • Copy the line of X.X snippet script tag.
  • Paste it to your HTML.
  • Test by if (window.jQuery) { alert('jQuery is loaded.') }
  • Usually use CDN way to load jQuery.

Syntax

  • $() (dollar sign) is a shortcut for jQuery(). Generally people use $() over jQuery().
