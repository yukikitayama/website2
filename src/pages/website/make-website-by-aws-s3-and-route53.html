Make website by AWS S3 and Route 53

Jun 29, 2021

Idea

  • Make a container to have website contents in AWS S3, and make it public to let everyone view your files there.
  • Buy a domain name in AWS Route 53 and configure the domain name goes to your S3 bucket. So people type your domain name in a browser to see your website.
  • For the cost, unless you store huge files, S3 is almost free. Route 53 monthly cost is $0.5 with an initial cost about $10. So minimum about $10 to $20 estimate.

S3

  • S3 is a product in AWS, something like a folder you can save files and data. We can configure who can access the data in S3 and how.
  • The reason for using S3 here is to make your website contents accessible to everyone anytime.

Route 53

  • Route 53 is also an AWS product where you can search whether a specific domain name is available, and if yes, you can buy it.
    • A domain name is a combination of certain characters and dot something like xxx.com or yyy.org.
  • After buying, you can configure where to go when people type your domain name in a web browser, to S3 here.

Steps

  • The following is a very beginner friendly, AWS official documentation about how to use S3 and Route 53 to make a website. If you just follow step by step, you can make it happen.
    • Configuring a static website using a custom domain registered with Route 53

Web design

  • S3 and Route 53 is kind of like a framework to run a website. But you need to prepare the actual contents, at least you need to store in S3 contents HTML files and styling CSS files.
  • To learn that, I took the following Web Design for Everybody courses in Coursera by University of Michigan. I think it was good.
    • Web Design for Everybody: Basics of Web Development & Coding Specialization

Example

  • This website is from S3 and Route 53.
  • I store files in S3, such as content page HTML files, styling CSS files, functions JavaScript files, and image JPG files.
  • I bought yukikitayama.com domain in Route 53, which was $12 for one time fee, and $0.5 monthly fee.
