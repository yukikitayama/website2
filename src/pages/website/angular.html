Setup

  • $ cd DIRECTORY_CONTAIN_PROJECTS
  • $ ng new PROJECT_NAME --no-strict
    • Change --no-strict mode later, but for getting started.
    • N to Would you like to add Angular routing
    • CSS, for the moment, to Which stylesheet format would you like to use?
  • $ cd PROEJCT_NAME
  • $ ng serve
    • Run development server, and you can see the website in a browser in http://localhost:4200/
    • Keep it running while developing. It automatically watches the project and rebuids new changes.

Use Bootstrap in Angular

  • $ npm install bootstrap
  • Open angular.json and update "styles" like the following.
    • Add "node_modules/bootstrap/dist/css/bootstrap.min.css", (for example) before "src/styles.css"
  • Rerun $ ng serve again.
  • Go to http://localhost:4200/, open Developer Console, Sources tab, style.css
  • Bootstrap is successfully added if you see Bootstrap in the comment section in style.css.
  • Now you can use Bootstrap classes in app component HTML files.

Component

  • Put everything in app folder. Make a subfolder whose name is equal to component names.
  • In selector you typically makes elements (like a tag), and don't use attribute ([COMPONENT]) and class selector (.COMPONENT).

Databinding

  • Databinding is communication between TypeScript code and the template HTML that users see.
  • Output data from TypeScript to template HTML
    • String interpolation by {{ data }}, or property binding by [property] = "data"
  • React to user events from template HTML to TypeScript
    • Event binding by (event) = "expression".
  • Combine both called Two-way-binding by [(ngModel)]="data"

Angular CLI

  • $ ng generate component COMPONENT_NAME
    • Create a template folder and files in app folder, and add import and declarations to app.moudle.ts.
