Memo from Coursera AWS DynamoDB

  • Leave "Use Lambda Proxy integration" unchecked.
  • Uncheck "Use Default Timeout" and set something in microsecond, meaning 10 seconds is 10000.
  • Cross-Origin Resource Sharing, CORS
    • Your website is hosted in one domain (your S3 website URL) and your API is hosted in a different domain (the API Gateway endpoint URL). Your browser doesn't like this, and blocks the request.
    • You need to let your browser know that it is OK to call your API Gateway endpoint URL from your website. You do this by using what is known as a preflight request.
    • This entire process of allowing cross-domain access with preflight requests is called "Enabling CORS"
  • Steps to enable CORS
    • Click Actions, select Enable CORS.
    • Check "DEFAULT 4XX" and "DEFAULT 5XX", and leave the other settings as they are.
