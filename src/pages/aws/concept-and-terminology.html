Decoupled architecture

  • Computing architecture that enables computing components or layers to execute independently while still interfacing with each other
  • Simple Queue Service (SQS)
    • Queues for storing messages while messages travel between application components, so it allows you to move data between distributed application components.
  • Simple Workflow Service (SWF)
    • A web service which coordinates work across distibuted application components.

Throughput

  • Measure how much data (how many packets) arrive at the destination successfully.
  • Unit to measure is data per second, or bits per second.
  • A high performance service has a high throughput.

Fanout

  • A message is published to an SNS topic. The message is replicated and pushed to multiple SQS queues. It allows for parallel synchronous processing.
  • In practice, assing SNS message filter policy to the topic subscription. The subscriber only receives a message that they are interested in.
